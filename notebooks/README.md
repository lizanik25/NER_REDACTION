# Ноутбуки с экспериментами

Ноутбуки содержат все этапы разработки — от сбора данных до оценки гибридной модели. 

## Содержание

| Ноутбук | Описание |
|---|---|
| `01_extract_real_texts.ipynb` | Извлечение текстов с PERSON из открытых корпусов (Nerus, NEREL, FactRuEval-2016, WiNER, Wikipedia, Mokoron Twitter). Фильтрация, дедупликация, отбор по форматам. |
| `02_synthetic_generation.ipynb` | Генерация синтетических текстов с PHONE, EMAIL, ADDRESS, ID с использованием Faker и шаблонов. Контроль распределения форматов и типов сущностей. |
| `03_mixed_text_generation.ipynb` | Формирование смешанных текстов: вставка синтетических ПД в реальные тексты с PERSON. Оформление в виде заявлений, анкет, договоров и пр. |
| `04_no_pii_dataset.ipynb` | Сбор выборки без персональных данных (210 текстов) из NEREL, Nerus, FactRuEval-2016, а также генерация 140 усложненных примеров — текстов, где фрагменты похожи на ПД, но не являются ими. |
| `05_corpus_annotation.ipynb` | Разметка корпуса: правила, конвертация в формат Doccano (JSONL), анализ итогового распределения сущностей по классам и источникам. |
| `06_slovnet_baseline.ipynb` | Оценка предобученной модели Natasha/Slovnet NER без дообучения. Перебор конфигураций (preprocessing × split_mode × batch_size). 
| `07_rule_based_experiments.ipynb` | Реализация и настройка rule-based детекторов на основе Yargy и регулярных выражений для PHONE, EMAIL, ID. |
| `08_slovnet_finetuning_and_hybrid.ipynb` | Дообучение Slovnet на собранном корпусе, построение гибридной модели, инференс на всех тестовых выборках. |



## Результаты экспериментов 

На основной тестовой выборке (300 текстов):

| Метод | micro-F1 | macro-F1 |
|---|---|---|
| Rule-based | 0.905 | 0.942 |
| Slovnet fine-tuned | 0.977 | 0.970 |
| **Hybrid** | **0.987** | **0.987** |
