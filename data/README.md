## Состав корпуса

Корпус сформирован из ~1900 текстов и содержит около 4000 размеченных вхождений сущностей пяти классов.


### Источники

Тексты с сущностями класса PERSON получены из открытых корпусов:

- **Nerus** - большой "серебряный" новостной корпус с NER-разметкой (PER, LOC, ORG) 
- **NEREL** - русскоязычный датасет для NER
- **FactRuEval-2016** - русскоязычный датасет для NER и фактов
- **WiNER** - корпус на основе русской Википедии (через Corus)
- **Wikipedia** - тексты из русской Википедии (через Corus)
- **Mokoron Russian Twitter Corpus** — тексты из Twitter (через Corus)
- **Synthetic** - синтетически сгенерированные примеры (Faker + шаблоны)

### Формат CSV-файлов

| Колонка | Описание |
|---|---|
| `source` | Источник получения текста |
| `text` | Текст |
| `label` | Разметка в формате JSONL-span: `[{"start": 0, "end": 10, "label": "PERSON"}, ...]` |
| `n_labels` | Число размеченных сущностей в тексте |
