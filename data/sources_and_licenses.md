# Источники данных и лицензии

## Открытые корпуса и датасеты

| Источник | Описание | Лицензия | Ссылка |
|---|---|---|---|
| **Nerus** | Открытый новостной корпус с автоматической разметкой NER (PER, LOC, ORG). Используется для извлечения текстов с PERSON и для выборки без ПД. | MIT | https://github.com/natasha/nerus |
| **NEREL** | Датасет для задачи NER на русском языке, содержит разметку именованных сущностей. Используется для текстов с PERSON и без ПД. | CC BY 4.0 | https://huggingface.co/datasets/iluvvatar/NEREL |
| **FactRuEval-2016** | Датасет  по извлечению фактов из русскоязычных текстов. Используется для текстов с PERSON и без ПД. | MIT | https://huggingface.co/datasets/gusevski/factrueval2016 |
| **WiNER** | Корпус на основе русской Википедии с разметкой именованных сущностей. Источник текстов с PERSON. | CC BY-SA 3.0 | (через Corus) |
| **Wikipedia (ru)** | Тексты из русской Википедии. Источник текстов с PERSON. | CC BY-SA 3.0 | (через Corus) |
| **Mokoron Russian Twitter Corpus** | Корпус текстов из Twitter на русском языке. Источник текстов с PERSON. | MIT | (через Corus) |

## Синтетические данные

Синтетические тексты сгенерированы с использованием:

- **Faker** (локализация `ru_RU`) - генерация имён, адресов, почтовых индексов, логинов. Лицензия: MIT. https://faker.readthedocs.io/en/master/locales/ru_RU.html
- **Python `random`** - выбор форматов и шаблонов.
- **Python `re`** - нормализация и фильтрация.


## Правовой статус

Все использованные корпуса опубликованы в рамках открытых научных исследований и не содержат персональных данных в смысле Федерального закона № 152-ФЗ «О персональных данных»:

- Датасеты NEREL и FactRuEval-2016 созданы для NLP-исследований, не направлены на идентификацию конкретных людей.
- Корпуса из каталога Corus собраны из публичных открытых источников.
- Nerus - открытый новостной корпус без ПД как таковых.

Обработка данных в данном проекте осуществляется в исследовательских целях и сопровождается анонимизацией там, где это необходимо.
